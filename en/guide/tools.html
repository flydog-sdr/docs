# Auxiliary tools

# Browser

FireFox

Mozilla Firefox, commonly known as Firefox, also commonly known as Firefox in Chinese, is a free and open source web browser developed by the Mozilla Foundation and its subsidiary Mozilla Corporation.

Chrome

Google Chrome is a free web browser developed by Google. Google Chrome itself is non-free software and is not fully open source.

# SD Card Burning Tool

balenaEtcher

balenaEtcher is a very easy to use removable disk image creation software, which provides users with a foolproof way to create removable disk images.

Rufus

Rufus is an open source tool that helps format and create bootable removable disks (Live USB) and is free for users to download and use.

Win32 Disk Imager

Win32 Disk Imager is an open source removable disk image creation and writing software.

# SSH Tools

Bitvise SSH

Bitvise SSH is a Windows client that supports SSH and SFTP. Developed and professionally supported by Bitvise. This SSH client is powerful, easy to install and easy to use.

PuTTY

PuTTY is a Telnet, SSH, rlogin, pure TCP and serial interface connection software. The whole software is less than 1M in size, but is fully functional and does not need to be installed, just download and use.

JuiceSSH

The best SSH tool for Android.

# Network Scan Tool

Advanced IP Scanner

Reliable and free network scanner that allows users to scan for online devices on their network segment and get information about the IP, hostname, MAC address, etc. of devices on the same LAN.

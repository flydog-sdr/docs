Quick start →

Awesome compatibility

The FlyDog SDR is compatible with KiwiSDR and runs smoothly on a variety of ARM based development boards such as Raspberry Pi, Orange Pi and others.

Affordable and high reliability

The FlyDog SDR is available at a much lower price with the same reliability as KiwiSDR.

Reverse proxy

The FlyDog SDR Project provides a dedicated reverse proxy server to penetrate the intranet for remote access to the FlyDog SDR and is configured with Content Delivery Network global acceleration.

Completely open source

From the core source to system image build scripts , code is all open source , do not leave any malware.

Proprietary update channel

The FlyDog SDR Project provides a proprietary update channel, which is tested multiple times before release to ensure availability.

Better performance

The FlyDog SDR uses 16-bit ADC for sampling and covers bandwidths up to 62 MHz with performance comparable to KiwiSDR.

# What is FlyDog SDR?

FlyDog SDR is an SDR product derived from KiwiSDR and uses 16-bit ADC sampling. It currently works smoothly on the full range of Raspberry Pi and Orange Pi models to meet a variety of scenarios.
